Canopy

Kit de démarrage pour vos projets web

Download this project as a .zip file Download this project as a tar.gz file

Canopy est un kit de démarrage pour vos nouveaux projets web.
Inspiré de SMACSS, il utilise Compass, Grunt & Bower.

Présentation

Scalable and Modular Architecture for CSS (SMACSS)

SMACSS est une méthode qui permet d'organiser votre CSS.
Elle permet notamment d'obtenir un code modulaire réutilisable et facilement maintenable sur vos projets web.
Plus d'informations sur la documentation officielle de SMACSS : http://smacss.com/

Installation

Pour utiliser Canopy, vous avez besoin d'installer Grunt, Compass et Bower.

Grunt

Grunt est un outil qui permet d'automatiser les tâches répétitives de votre projet.
Plus d'informations sur la documentation officielle de grunt.js : http://gruntjs.com/

Grunt utilise Node.js, NPM et Grunt CLI qui permet de gérer Grunt en ligne de commande.

Node.js et NPM

Pour ceux qui utilisent Homebrew, il vous suffit de rentrer la commande suivante dans votre terminal :

$ brew install node
Grunt CLI
$ npm install -g grunt-cli

Compass

Compass est un framework CSS basé sur le langage Sass, qui propose un ensemble de mixins pour vos projets.
Plus d'informations sur la documentation officielle de Compass : http://compass-style.org/help/

$ gem install compass

Bower

Bower est un outil qui permet de gérer les dépendances de votre projet.
Plus d'informations sur la documentation officielle de Bower : http://bower.io/

$ npm install -g bower

Utilisation

Télécharger Canopy puis lancer sur votre terminal

$ grunt
